# MVC Project 

## Project Overview
This project demonstrates a **secure user authentication and role-based authorization system** built using the **MVC (Model-View-Controller) architecture**.  
It allows users to register, login, and access protected routes based on their roles (Admin/User).

---



---

## Features
- User Registration & Login (Authentication)
- Role-Based Authorization (Admin / User)
- JWT Token-Based Authentication
- Password Hashing using bcrypt
- Protected routes
- Logout functionality
- Environment variables for sensitive data
